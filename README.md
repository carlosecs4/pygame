**INSPER FIGHT**
Projeto feito pelos alunos: Mateus Pratas, Carlos Sousa e Lorenzo Stabile

Para rodar certifique-se que tenha instalado python 3.10+ e a biblioteca "pygame". 

**Para exectuar o jogo**

1. Baixe o repositório
2. Verifique se todas as pastas e arquivos foram baixados na mesma pasta
3. Execute o arquivo "jogo.py"

Vídeo explicando como jogar Insper Fight: https://www.youtube.com/watch?v=LTYZVN19b9A
